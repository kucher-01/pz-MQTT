# Розгортання та налаштування MQTT-брокера

## Мета
Розгорнути MQTT-брокер (Mosquitto) за допомогою Docker, налаштувати базову конфігурацію та продемонструвати обмін повідомленнями (Publish/Subscribe).

## Запуск проєкту
1. Перейдіть до директорії `broker`.
2. Виконайте команду для запуску брокера:

```bash
   docker compose up -d
```

Брокер буде доступний на localhost:1883
